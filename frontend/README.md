# AI 穿搭推荐系统 - 前端项目

## 项目简介

这是一个基于 HTML + CSS + JavaScript 开发的 AI 穿搭推荐系统前端页面，包含温度查询、穿搭推荐和虚拟试衣三个核心功能模块。

## 功能特点

1. **温度查询模块**：输入城市名查询实时温度、体感温度、天气类型、湿度和风速
2. **穿搭推荐模块**：根据场景和温度生成个性化穿搭推荐，展示推荐衣物列表和搭配理由
3. **虚拟试衣模块**：支持上传人物照片和衣物照片，生成虚拟试衣效果
4. **响应式设计**：适配 PC 端和移动端设备
5. **友好的用户体验**：加载动画、成功/失败提示等

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- 原生 Fetch API（用于后端接口调用）

## 后端接口

本项目需要与以下后端接口配合使用：

- 服务地址：http://localhost:8000
- 健康检查：GET /health
- 天气查询：POST /api/weather/query → 入参 {"location":"城市名"}
- 穿搭推荐：POST /api/outfit/recommend → 入参 {"user_id":123,"location":"城市名","scene":"场景"}
- 虚拟试衣：POST /api/virtual-tryon → 入参 {"person_image_url":"URL","garment_image_url":"URL","garment_type":"top/bottom/dress"}

## 运行说明

### 方法一：直接打开 HTML 文件

1. 确保后端服务已启动并运行在 http://localhost:8000
2. 在浏览器中直接打开 `index.html` 文件
3. 开始使用系统功能

### 方法二：使用本地服务器（推荐）

1. 确保后端服务已启动并运行在 http://localhost:8000
2. 在前端目录下启动本地服务器，例如：

   ```bash
   # 使用 Python 3
   python -m http.server 8080
   
   # 或使用 Node.js
   npx http-server -p 8080
   ```

3. 在浏览器中访问 http://localhost:8080
4. 开始使用系统功能

## 使用指南

1. **温度查询**：
   - 在输入框中输入城市名（默认北京）
   - 点击「查询温度」按钮
   - 查看实时温度、体感温度、天气类型、湿度和风速

2. **穿搭推荐**：
   - 选择场景（商务会议/休闲约会/日常通勤/运动健身）
   - 确认用户ID（默认123）
   - 点击「生成穿搭推荐」按钮
   - 查看推荐衣物列表和搭配理由

3. **虚拟试衣**：
   - 上传人物照片（支持URL或本地上传）
   - 上传衣物照片（支持URL或本地上传）
   - 选择衣物类型（上装/下装/连衣裙）
   - 点击「生成试衣效果」按钮
   - 查看生成的虚拟试衣图片

## 注意事项

1. 确保后端服务已启动并运行在 http://localhost:8000
2. 图片上传功能会将图片转换为 Base64 格式，较大的图片可能会影响性能
3. 虚拟试衣功能使用阿里云 API，需要确保网络连接正常
4. 本项目使用了模拟数据，实际使用时需要替换为真实的后端接口
